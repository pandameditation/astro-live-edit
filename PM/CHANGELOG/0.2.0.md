# Version 0.2.0

**Feature:** MutationObserver-based change detection in `/public/live-edit.js`
- Adds MutationObserver to all contenteditable elements (watches `childList`, `characterData`, `subtree`)
- 500ms debounce prevents excessive state updates
- Save button shows yellow glow indicator when unsaved changes exist
- Blur event handling preserved alongside MutationObserver (complementary, not replacement)
- `_suppressObserver` flag prevents false positives during Tab/Shift+Tab list indentation

**⚠️ IMPORTANT — Developer Warning:**
The MutationObserver fires on ALL subtree mutations inside contenteditable elements. If a future feature adds programmatic DOM changes, it MUST set `_suppressObserver = true` around them (like the Tab handler does) to prevent false change detection.

**Files Changed:**
- `public/live-edit.js` — MutationObserver, debounce, visual indicator, trackChange extraction
